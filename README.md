# 📈 Finance Dashboard

A modern and interactive Finance Dashboard built using React. It helps users track their income, expenses, and overall financial health with clean UI and data visualization.

---

## 🚀 Features

- 💰 Track Income & Expenses
- ✏️ Add, Edit, and Delete Transactions
- 📊 Interactive Charts (Pie & Line)
- 📅 Monthly Financial Summary
- 🌙 Dark Mode Support
- 👤 Role-based Access (Viewer / Admin)
- 💾 Data Persistence using Local Storage
- 📤 Export Data as JSON

---

## 🛠️ Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 📊 Recharts
- 💾 Local Storage

---

## 📦 Installation & Setup

Follow these steps to run the project locally:

# Clone the repository

git clone https://github.com/ankit-0x/Finance-Dashboard.git

# Go to project folder

cd finance-dashboard

# Install dependencies

npm install

# Start development server

npm run dev
